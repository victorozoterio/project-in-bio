<h2 id="description">📚 Description</h2>

This is a micro SaaS developed to facilitate the organization and storage of personal, professional and portfolio projects and links in one place. The system allows quick and secure access through Google login and offers a 3-day free trial, with monthly subscription or lifetime access options.

The project was developed using **Next.js**, taking advantage of its ability to create APIs directly, and integrates **Firebase** as a database. In addition, the platform has a payment system via **Stripe**, allowing the management of subscriptions in a secure and efficient way.

Main features:
- 📌 Store and organize your projects and links easily.
- 🔐 Fast and secure login with Google.
- 💳 Managed subscription via Stripe.
- 🚀 Backend using Next.js Routes API.
- 🔥 Firebase for data storage.

Feel free to test and send any feedback! 💗

<h2 id="installation">⚙️ Installation</h2>

- 1: Clone this repository: `git clone https://github.com/victorozoterio/project-in-bio.git`;
- 2: Create a `.env` file from the `.env.example` file;
- 3: Fill in all the necessary variables in the `.env`;
- 4: Install the dependencies, running the command: `npm install`;
- 5: Run the application, running the command: `npm run dev`
